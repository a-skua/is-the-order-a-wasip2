## 第2章 サンプルコード

### WATの実装例

```sh
cd calc-wat
make
```

### Rustの実装例

```sh
cd calc
make
```

### Goの実装例

[tinygo](https://tinygo.org/)が必要です．

```sh
cd calc-go
make
```

### JSの実装例

[node](https://nodejs.org/)が必要です．

```sh
cd calc-js
make
```

### Wasmコンポーネントの利用

```sh
cd run
make
```

#### Go，JS，WAT製のコンポーネントを利用する

```sh
# WAT
make run-wat

# Go
make run-go

# JS
make run-js
```
